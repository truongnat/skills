# Docs

Tài liệu repo-level được gom vào đây để tránh rải Markdown ở root.

## Nội dung

- `architecture.md`: kiến trúc và workflow tổng thể
- `design-system-astryx.md`: baseline design system cho HTML preview, tool UI và artifact UI
- `agents-readme.md`: nội dung copy vào `.agents/README.md`
- `sessions-readme.md`: nội dung copy vào `.agents/sessions/README.md`

## Quy ước

- `AGENTS.md` vẫn giữ ở root vì đây là contract chính cho agent.
- Tài liệu dùng để package/install hoặc làm canonical reference nên ưu tiên đặt trong `docs/`.

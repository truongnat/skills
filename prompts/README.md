# Prompts

Đặt các **file Markdown** (`.md`) mô tả prompt có thể tái sử dụng, ví dụ:

- `templates/` — một file một “template” với mục System / User và biến `{{tên_biến}}`
- `chains/` — (tùy chọn) chuỗi bước mô tả bằng Markdown, tương tự [workflows/](../workflows/)

Thư viện ví dụ và định dạng mô tả: [templates/PROMPT_TEMPLATES.md](../templates/PROMPT_TEMPLATES.md) (định dạng thể hiện bằng Markdown, không bắt buộc file `.yaml`/`.yml`).

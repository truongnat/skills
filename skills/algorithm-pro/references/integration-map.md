# Algorithm — integration map

| Skill | Combine when |
|-------|----------------|
| **`testing-pro`** | Property-based tests, adversarial inputs, regression suite for known bugs. |
| **`data-analysis-pro`** | Plotting runtimes, distribution of inputs, A/B of implementations. |
| **`typescript-pro` / `javascript-pro`** | Language-specific numeric limits, BigInt, typed arrays. |
| **`postgresql-pro`** | When “algorithm” is actually query + index design — delegate SQL side. |
| **`network-infra-pro`**, **`deployment-pro`**, **`caching-pro`** | Distributed execution, reliability, memoization at system boundary — pair with **`algorithms-in-systems.md`** |

**Boundary:** **`algorithm-pro`** owns modeling, strategy, correctness/complexity, and algorithmic patterns; **system integration** (queues, shards, SLAs) is shared with stack skills.

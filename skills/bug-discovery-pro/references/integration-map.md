# Bug discovery — integration map

| Skill | When |
|-------|------|
| **`testing-pro`** | Repro, coverage, flaky CI, golden tests, regression suite after fix. |
| **`security-pro`** | Vuln class, trust boundaries, abuse cases — graph does not prove security. |
| **`content-analysis-pro`** | Parse logs, crash dumps, build output as attached text. |
| **`api-design-pro`** | Contract drift, idempotency/retry semantics, cross-boundary errors. |
| **`deployment-pro`** | Rollouts, mixed versions, feature flags, platform incidents. |
| **`network-infra-pro`** | Topology, saturation, cascading failures, DNS/LB. |
| **GitNexus Cursor skills** | `gitnexus-cli`, `gitnexus-impact-analysis`, `gitnexus-debugging`, `gitnexus-exploring` — indexing and graph depth. |

**Boundary:** **`bug-discovery-pro`** owns **methodology**, **taxonomy-informed triage**, **graph-assisted discovery**, and **telemetry-aware** investigation narratives; **fixes** live in domain **`*-pro`** skills and **runtime** execution is often owned by **platform/on-call** playbooks.

# Business analysis — integration map

| Skill | When |
|-------|------|
| **`planning-pro`** | Phasing, dependencies, roadmap sequencing, effort/risk bands. |
| **`strategic-consulting-pro`** | Portfolio prioritization, exec narrative, multi-program trade-offs. |
| **`security-pro`** | Abuse cases, data classification, compliance language, control mapping. |
| **`testing-pro`** | Acceptance criteria as testable checks; experiment validation quality. |
| **`data-analysis-pro`** | Metric definitions, baselines, cohort analysis, evidence for KPI claims. |
| **`content-analysis-pro`** | Deep extraction from PDFs, media, contracts before BRD traceability. |
| **`api-design-pro`** | Latency/reliability contracts, evolution constraints feeding NFRs. |
| **`deployment-pro`** | Release model, environments, rollout constraints in NFRs. |
| **Stack `*-pro`** | Feasibility spikes, architecture limits, order-of-magnitude cost. |

**Boundary:** **`business-analysis-pro`** owns **problem framing**, **decision-quality artifacts**, **requirements traceability**, and **outcome-aligned metrics**; engineering skills own **implementation truth**.

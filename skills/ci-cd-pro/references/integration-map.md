# CI/CD — integration map

| Skill | When |
|-------|------|
| **`docker-pro`** | Image build, layer cache, BuildKit in workflow. |
| **`deployment-pro`** | Promotion, canary, rollback after CI green. |
| **`testing-pro`** | Job order, flakes, coverage artifacts. |
| **`security-pro`** | OIDC, pinned actions, fork PR policy. |
| **`git-operations-pro`** | Triggers, protected branches, tags. |
| **`code-packaging-pro`** | Publish wheels/containers from pipeline. |

# Content analysis — decision tree

## Input type

- **Structured data** (CSV, JSON) → Schema first; sampling strategy.
- **Unstructured** (docs, media) → Scope by corpus slice; define unit of analysis.

## Goal

- **Summarization** vs **classification** vs **compliance scan** — pick method and rubric before reading bulk.

## Scale

- **Small corpus** → Manual + spot checks.
- **Large** → Scripted extraction + validation subset; **`data-analysis-pro`** for tabular stats.

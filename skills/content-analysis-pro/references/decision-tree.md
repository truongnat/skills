# Content analysis — decision tree

## Input type

```
Structured tabular (CSV, Parquet)?
├── Schema + stats path — pair data-analysis-pro for aggregates
└── Narrative PDF / slides / media?
    └── Multimodal pipeline — analysis-pipeline-system-model.md
```

## Goal

```
Summarize only?
├── Executive layer + optional deep dive — decision-framework-and-tradeoffs.md
├── Extract entities / dates / obligations?
│   └── Schema-first; cite every row — evidence-and-provenance-rules.md
└── Compare sources?
    └── Same depth per source; conflicts explicit
```

## Scale / coverage

```
Whole document / full video feasible?
├── Read fully with anchors
└── Too long for one pass?
    └── Stratified sample + **disclosed rule** — OR staged continuation — edge-cases.md
```

## Risk domain

```
Legal / medical / financial obligations detected?
├── Assistive analysis only — human expert required — reporting-and-limitations.md
└── General business content?
    └── Standard evidence map — still no fabricated citations
```

## Grounding strictness

```
Litigation / audit traceability?
├── Verbatim quotes + page; tool versions if automated — evidence-and-provenance-rules.md
└── Informal review?
    └── Still no invented facts — quality-validation-and-guardrails.md
```

# Evidence and provenance rules

## Minimum viable citation

| Modality | Anchor |
|----------|--------|
| **PDF / doc** | Page number or section heading + short quote |
| **Spreadsheet / CSV row** | Row index or key column value (if stable) |
| **Image** | Region description (“top-right banner”) or crop reference |
| **Video** | `hh:mm:ss`–`hh:mm:ss` or single timestamp + what was visible/heard |
| **Audio / transcript** | Line or time range |

## Quote discipline

- **Short** excerpts for verification; long blocks only if user requests or fair-use clear.
- **Ellipsis** (`…`) for omissions; **\[sic\]** for source errors if quoting formally.

## Chain of reasoning

For comparisons: **A says** / **B says** — avoid merged “truth” unless sources agree.

## Tool provenance

If FFmpeg/OCR/API used: **name + purpose** in methods footnote — supports audit — **`versions.md`**.

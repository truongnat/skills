# Content analysis tips and tricks

## Before analysis

- Ask for **goal** and **constraints** (length of report, language, redaction rules).
- If multiple files — **order** (chronological, priority) and **relationship** (appendix vs main).

## During analysis

- **Start with structure** — Outline first, then fill detail (reduces missed sections).
- **Named entities** — Consistent spelling across report; align with source.
- **Numbers** — Double-check **units** and **currency** from image/table.

## Video

- **Sample strategy** — For long video, segment by **chapters** or **fixed intervals** if full frame-by-frame is impractical; state **coverage** (“reviewed first 15 min in detail”).

## Images

- **Crop context** — When user sends a crop, note **missing surrounding context** that could change meaning.

## Documents

- **Scanned PDFs** — OCR errors common; flag **suspected** misreads (`[unclear: …]`).

## After analysis

- **Actionable** — Turn findings into **next questions** or **checklist** for the user.

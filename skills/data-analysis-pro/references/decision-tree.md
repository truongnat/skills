# Data analysis — decision tree

## Data size and tool

- **Fits memory** → pandas/polars; profile dtypes early.
- **Out of core** → Chunking, SQL pushdown, or big-data stack (out of this skill’s core).

## Question type

- **Descriptive** → EDA + tables first.
- **Inferential** → State assumptions, sample design, confidence intervals.

## Clean vs explore

- **Dirty ingest** → Cleaning checklist before modeling claims.

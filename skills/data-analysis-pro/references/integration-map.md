# Data analysis — integration map

| Skill | When |
|-------|------|
| **`sql-data-access-pro`** | Parameterized SQLite extracts; SQL hygiene before pandas |
| **`postgresql-pro`** | Pushdown aggregates, indexing, warehouse-scale patterns |
| **`content-analysis-pro`** | Qualitative coding of free text — not pivot tables alone |
| **`business-analysis-pro`** | KPI definitions, roadmap narrative from metrics |
| **`market-research-pro`** | External sizing assumptions validation |
| **`security-pro`** | Anonymization, safe exports, notebook redaction |
| **`testing-pro`** | Golden checks for analytical pipelines / regression tests on transforms |
| **`algorithm-pro`** | Heavy algorithmic complexity or complexity proofs — rare for standard EDA |
| **`repo-tooling-pro`** | Batch scripts, repeatable pipeline wiring |

**Boundary:** **`data-analysis-pro`** owns **tabular computation, visualization discipline, spreadsheet delivery**; **`postgresql-pro`** owns **live database** design/ops beyond ad-hoc extracts.

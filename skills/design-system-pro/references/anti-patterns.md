# Design system — anti-patterns

## One-off hex values

- Inconsistent UI debt.
- **Fix:** Tokenize; document exceptions.

## Color-only status

- WCAG and color-blind failure.
- **Fix:** Icon + text + pattern.

## Trend-first clutter

- Glassmorphism on dense tables without contrast check.
- **Fix:** Judgment-first; test real content.

## Web patterns on native

- Wrong back behavior and touch targets.
- **Fix:** Platform-specific guidance + **`mobile-design-pro`**.

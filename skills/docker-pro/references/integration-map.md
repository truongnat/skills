# Docker — integration map

| Skill | When |
|-------|------|
| **`ci-cd-pro`** | Build/push images in pipelines, cache, matrix. |
| **`code-packaging-pro`** | `pyproject` + Dockerfile for Python services. |
| **`deployment-pro`** | Registry, rollout, K8s/Cloud Run targets. |
| **`security-pro`** | Scanning, non-root, secrets, `--secret` BuildKit. |
| **`network-infra-pro`** | Compose networks, DNS, overlay vs bridge. |
| **`testing-pro`** | Testcontainers, integration tests against real DB/message brokers in CI. |
| **`postgresql-pro`** | DB container env, persistence, migration containers paired with app image. |

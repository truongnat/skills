# Edge cases

- **Conflicting reviewer feedback**: recommendations disagree and create rework loops.
- **Low-confidence evidence**: issue suspected but not reproducible yet.
- **Deadline pressure**: must defer lower-severity items safely.
- **Cross-file coupling**: fix in one area can regress another subsystem.
- **Spec ambiguity**: unclear requirement makes correctness judgment unstable.
- **Feedback overload**: too many comments reduce focus on critical issues.

# Flutter — integration map

| Combined skill | Why | `flutter-pro` owns | Other skill owns |
|----------------|-----|---------------------|-------------------|
| **`testing-pro`** | Golden / integration | `integration_test`, driver setup | CI strategy |
| **`deployment-pro`** | Store / web deploy | Build flavors, signing handoff | Server infra |
| **`security-pro`** | Data at rest | `flutter_secure_storage` patterns | API auth |
| **`design-system-pro`** | Tokens in Flutter | `ThemeData`, component mapping | Figma source |
| **`sql-data-access-pro`** | Local SQLite | Drift/sqflite usage | SQL schema |
| **`ci-cd-pro`** | Flutter build in CI | `flutter test`, flavors, caching | Workflow YAML |
| **`mobile-design-pro`** | Touch/safe-area UX depth | Widget implementation | Platform HIG references |

**Handoff:** Platform channel or FFI work with native teams — document contract in Dart + native stubs.

# Image processing — integration map

| Skill | When |
|-------|------|
| **`content-analysis-pro`** | Semantic meaning, OCR, charts — not pixel transforms alone. |
| **`security-pro`** | EXIF/GPS strip, redaction before publish, untrusted uploads. |
| **`deployment-pro`** | Asset pipeline in CI, CDN cache headers. |
| **`testing-pro`** | Golden-file / snapshot tests for image outputs. |
| **`docker-pro`** | Headless image builds; codec/font deps in containers. |
| **`web-research-pro`** | License/source for stock or reference images. |

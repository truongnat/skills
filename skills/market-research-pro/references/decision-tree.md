# Market research — decision tree

## Question type

- **TAM/SAM/SOM** → State assumptions; show sensitivity.
- **Competitive positioning** → Criteria matrix + evidence links.

## Data availability

- **No public data** → Triangulate proxies; label uncertainty bands.

## Output audience

- **Exec** → one-page thesis + appendix evidence.
- **Product** → segment priorities + feature implications.

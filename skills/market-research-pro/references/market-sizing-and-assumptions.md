# Market sizing and assumptions

## Core methods

- **Top-down**: start from industry totals, then filter by geography, segment, and use case.
- **Bottom-up**: estimate reachable accounts x expected ARPA/adoption.
- **Value-theory**: size by economic value created/captured (useful in emerging categories).

## TAM / SAM / SOM

- **TAM**: total theoretical demand in the defined category.
- **SAM**: portion that matches current offering constraints.
- **SOM**: realistic short/mid-term share given channel, brand, and sales capacity.

## Assumption discipline

- Keep each assumption explicit: source, date, rationale, confidence.
- Run sensitivity with low/base/high scenarios.
- Avoid mixing year ranges or currencies without normalization.

## Output expectation

- One table for assumptions and one table for scenario results.
- Include confidence notes and key unknowns.

# Edge cases

- **Scope churn**: requirements shift mid-phase and invalidate downstream tasks.
- **Cross-team dependency delay**: another team misses delivery window and blocks critical path.
- **Unknown unknowns**: legacy systems reveal hidden constraints late.
- **Resource volatility**: key owner unavailable during critical milestones.
- **Partial completion trap**: many tasks are started but few are done, hiding schedule risk.
- **Over-optimization**: planning too deeply for uncertain phases wastes time.

# Security standards — versions (reference)

`security-pro` is not tied to a runtime version — use this table for **external standards** to cite in reviews.

| Standard | Role |
|----------|------|
| **OWASP Top 10** | Risk classes for web/API |
| **OWASP ASVS** | Verifiable controls by level (1–3) |
| **CWE** | Weakness enumeration for findings |
| **MITRE ATT&CK** | Defensive mapping for detection coverage |

Confirm org policy (PCI, HIPAA, SOC2) separately from framework version.

# Edge cases

See also thematic tables — **`failure-modes-detection-mitigation.md`**.

- **Metric gaming**: local metric improves while real outcome quality drops.
- **Sparse feedback**: too little signal to validate intervention impact.
- **Confounded changes**: many simultaneous edits hide true cause/effect.
- **Overfitting to benchmark**: gains do not generalize to production tasks.
- **Reviewer inconsistency**: shifting standards produce noisy conclusions.
- **Delayed failures**: regressions appear long after intervention rollout.

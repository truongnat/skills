# Self-improve agent — integration map

| Skill | When |
|-------|------|
| **`feedback-pro`** | Structured review signals, severity, evidence quality. |
| **`planning-pro`** | Phased improvement waves, milestones, capacity. |
| **`testing-pro`** | Regression suites, scenario tests, eval harness in CI. |
| **`repo-tooling-pro`** | Batch metrics, `analyze-skills`, repo audit automation. |
| **`skills-self-review-pro`** | Meta-quality for bundled skills and authoring rules. |
| **`git-operations-pro`** | Branch/PR hygiene for harvested skill contributions. |
| **`security-pro`** | Human gate and threat review when agents touch sensitive data or tooling. |
| **`bug-discovery-pro`** | Systematic repro when failures are intermittent or environmental. |

**Handoff:** **`self-improve-agent-pro`** owns the **improvement loop design**; paired skills supply **signals**, **plans**, **tests**, and **shipping** discipline.

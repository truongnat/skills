# SEO — anti-patterns

## Keyword stuffing / thin pages

- Doorway clusters; manual action risk.
- **Fix:** Intent-aligned content; consolidate duplicates.

## Invisible structured data

- JSON-LD not matching visible content.
- **Fix:** Guidelines-compliant markup only.

## Ten changes at once

- Cannot attribute GSC moves.
- **Fix:** One lever per deploy window where possible.

## Staging indexed

- Duplicate/thin prod risk.
- **Fix:** `noindex`, auth, IP allowlist; robots on staging.

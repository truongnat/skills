# SEO — decision tree

## Channel

- **Organic is primary** → Full lifecycle + GSC discipline.
- **Organic is secondary** → Technical baseline only; avoid over-investment.

## Rendering

- **SSR/SSG** → Crawlable HTML default path.
- **Heavy SPA** → Prerender/hybrid or documented crawl risk.

## Duplicate URLs

- **Facets/parameters** → Canonical policy or noindex low-value combos first.

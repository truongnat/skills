# SEO tips and tricks

Practical patterns that compound when applied **consistently** at template level.

## On-page (templates)

- **Title** — Unique per page; primary intent near the front; avoid boilerplate stuffing.
- **Meta description** — Persuasive snippet (not a ranking lever alone); align with **search intent**.
- **H1** — One clear H1; hierarchy **H2–H3** reflects outline (not for keyword stuffing).
- **Internal links** — Descriptive anchor text; hub pages for topic clusters; fix **orphan** URLs.

## Content

- **Intent match** — Page type matches query (guide vs product vs comparison).
- **E-E-A-T** — Experience, expertise, authoritativeness, trust — especially for YMYL topics; author, citations, freshness where relevant.
- **Thin content** — Merge or improve vs many weak URLs competing for same intent.

## Snippets and SERP

- **FAQ / HowTo** schema — Only if content is **visible** and accurate; abuse triggers quality issues.
- **Sitelinks** — Influenced by structure and branded strength; not directly “requested.”

## Images and media

- **Descriptive file names and alt** where helpful for accessibility and context; avoid keyword spam.
- **Lazy load** — Do not lazy-load **above-the-fold** hero (hurts LCP).

## Measurement

- **Google Search Console** — Queries, pages, index coverage, **manual actions**, Core Web Vitals report.
- **Segment** — Branded vs non-branded; landing page groups; before/after deploy dates.

## Low-effort wins

- Fix **soft 404** and **redirect loops**; resolve **duplicate** titles across pagination/filters.
- **Pagination** — `rel="next/prev"` deprecated by Google but **clear** crawl paths still matter — use consistent linking.

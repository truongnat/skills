# SQL data access — integration map

| Skill | Combine when |
|-------|----------------|
| **`postgresql-pro`** | Server Postgres, migrations, `EXPLAIN (ANALYZE)`, RLS, ops. |
| **`data-analysis-pro`** | Post-export pandas, charts, validation of CSV/Parquet. |
| **`security-pro`** | Injection in apps, credential storage, least privilege. |
| **`nestjs-pro` / others** | ORM layer — this skill focuses on raw SQLite/stdlib patterns. |

**Boundary:** `sql-data-access-pro` = portable SQL habits + SQLite; production PG is **`postgresql-pro`**.

# Strategic consulting — anti-patterns

## Analysis paralysis

- 50 slides, no decision.
- **Fix:** Time-box; minimum viable recommendation.

## Hidden options

- Only one “obvious” path presented.
- **Fix:** At least two credible alternatives with trade-offs.

## Strategy without execution link

- No owner, metric, or date.
- **Fix:** Initiative map + KPI + **`planning-pro`** handoff.

## Jargon wall

- Exec cannot act.
- **Fix:** Bottom-line summary + appendix depth.

# Strategic consulting — decision tree

## Decision type

- **Go/no-go** → Criteria, options, recommendation, risks.
- **Portfolio** → Capacity, sequencing, kill criteria.

## Uncertainty

- **High volatility** → scenarios (base/downside/upside) not single forecast.

## Stakeholder alignment

- **Conflict** → RACI + decision log before deep analysis.

## Strategy → execution handoff

```
Chosen option needs phased delivery, dependencies, and milestones?
├── Yes → explicit **`planning-pro`** handoff after decision log — **`strategic-decision-and-stakeholder-system-model.md`**
└── Single decision only → still name owner, metric, and review date
```

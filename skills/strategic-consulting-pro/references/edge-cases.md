# Edge cases

See thematic mitigations — **`failure-modes-detection-mitigation.md`**.

- **Regulated or security-constrained bets** — Data residency, breach exposure, or sector rules may veto “obvious” growth moves — align with **`security-pro`** when technical policy gates strategy.
- Stakeholder conflict: recommendation must separate evidence from politics.
- Data scarcity: use proxy assumptions and clearly mark confidence.
- Strategy drift: revisit decisions when trigger metrics cross thresholds.
- Over-planning: avoid long strategy decks without executable next steps.

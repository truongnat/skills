# Strategic consulting — integration map

| Skill | When |
|-------|------|
| **`market-research-pro`** | Market sizing, competitor structure, segment attractiveness. |
| **`web-research-pro`** | Source verification, recency, and quality of external claims. |
| **`data-analysis-pro`** | Quantitative baselines, scenario math, KPI evidence. |
| **`business-analysis-pro`** | Requirements, scope, and acceptance criteria from chosen strategy. |
| **`planning-pro`** | Roadmaps, sequencing, dependencies, and delivery checkpoints after decision. |
| **`feedback-pro`** | Stakeholder review cycles and prioritization from review findings. |
| **`security-pro`** | Strategy constrained by data protection, threat landscape, or regulated posture. |
| **`auth-pro`** | Identity / access strategy depth for build-partner-buy in auth space. |

**Handoff:** **`strategic-consulting-pro`** frames **options and decision**; **`planning-pro`** converts to **executable plan** when delivery complexity warrants.

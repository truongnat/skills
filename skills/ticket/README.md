# Ticket skill — đã chuyển sang global

Skill **ticket** (kanban / `ticket:start` … `ticket:feedback`) không còn bản sao đầy đủ trong repo này.

**Bản chính (dùng mọi workspace):**

`%USERPROFILE%\.cursor\skills\ticket\`

- Entry: `SKILL.md`
- Tham chiếu: `references/`, `README.md`

Artifact ticket vẫn nằm **trong từng project** tại `/.kanban/tickets/<ticket>/` (theo `SKILL.md`).

Để cập nhật skill cho tất cả project: sửa file trong thư mục global trên (hoặc đồng bộ từ bản backup của team).

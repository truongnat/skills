# Integration map — typescript-pro

| Combined skill | Why | `typescript-pro` owns | Other skill owns |
|----------------|-----|-------------------------|------------------|
| **`react-pro`** | TSX | Props, children, event types | Runtime behavior, hooks |
| **`nestjs-pro`** | DI / decorators | Typing providers and DTOs | Nest APIs, modules |
| **`testing-pro`** | Test types | Mock typing, helper generics | Runners, strategy |
| **`javascript-pro`** | Runtime semantics | When behavior is JS (event loop, prototypes) not types | TypeScript types and `tsc` |

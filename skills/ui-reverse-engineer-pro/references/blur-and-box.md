# Blur & Box — Advanced Technique

This is a powerful technique for seeing structure without visual noise.

## The Technique

Look at the image at a smaller size, or imagine blurring the image, keeping only the large blocks.

## How to Apply

### Method 1: Scale down
- View image at 25-50% size
- Details disappear, blocks remain
- Focus on structure, not pixels

### Method 2: Mental blur
- Imagine the image is blurred
- Ignore colors, icons, fine details
- See only major blocks

### Method 3: Squint
- Squint your eyes at the image
- Fine details blur away
- Large shapes remain visible

## What to Look For

After applying blur/box, ask:

- Which regions remain?
- Which regions have visual weight?
- Which regions are secondary?
- How many major blocks are there?
- What is the layout direction?

## Purpose

This technique helps you:

- Remove visual noise
- See blocks clearly
- Identify spacing
- Understand hierarchy
- Focus on structure over details

## Example

**Original image:**
- Complex gradient background
- Multiple icons
- Fine text details
- Subtle shadows
- Decorative elements

**After blur/box:**
- 3 main vertical sections
- Hero section at top
- 2-column grid in middle
- Full-width section at bottom
- Clear vertical rhythm

## When to Use

- Complex images with lots of visual noise
- When you're getting distracted by colors/details
- When layout structure is unclear
- When you need to identify major blocks quickly
- When analyzing competitor UIs for patterns

## Combination with L.A.S.T.C

Use Blur & Box before applying L.A.S.T.C:

1. **Blur & Box** — Identify major blocks
2. **L** — Confirm layout structure
3. **A** — Check alignment of blocks
4. **S** — Estimate spacing between blocks
5. **T** — Identify text hierarchy (if visible)
6. **C** — Name the blocks as components

This sequence gives you structure first, then details.

# Web research — anti-patterns

## Single SEO blog as authority

- Copy-paste errors propagate.
- **Fix:** Cross-check primary source.

## Undated claims

- “Latest” without version.
- **Fix:** Lock doc revision date or git tag.

## Paywalled source without note

- Unverifiable for reader.
- **Fix:** Secondary summary + label uncertainty.

## Scraping ToS violation

- Legal/ops risk.
- **Fix:** Use official API or allowed export.

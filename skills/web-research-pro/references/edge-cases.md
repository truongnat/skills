# Edge cases

**Related:** verification model — [web-research-verification-system-model.md](web-research-verification-system-model.md), [failure-modes-detection-mitigation.md](failure-modes-detection-mitigation.md).

- **Conflicting** sources — prefer **newer** official doc **or** reproduce with a minimal **snippet** in the repo.
- **SEO spam** — thin pages that mirror docs; verify on **official** property.
- **AI-generated** summaries on the web — **verify** against primary docs.
- **Regional** blocking — some doc CDNs geo-block; try alternate mirrors or vendor docs.

# artifact-bundle

Gom nhiều artifact vào một preview portal nhỏ.

## Entry

`index.ts`

## Cách dùng

```bash
node dist/tools/artifact-bundle/index.js --file PLAN.md --file output.json --open
```

## Mô tả nhanh

- show nhiều file cùng lúc
- mỗi card có nút select để emit path file
- phù hợp để review nhiều artifact trong một màn hình

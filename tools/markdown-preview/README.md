# markdown-preview

Preview Markdown thành HTML dễ đọc.

## Entry

`index.ts`

## Cách dùng

```bash
node dist/tools/markdown-preview/index.js --file README.md --open
```

## Mô tả nhanh

- nhận Markdown từ file, inline, hoặc stdin
- render heading, list, quote, code block
- phù hợp cho plan, review, report, docs

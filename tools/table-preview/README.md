# table-preview

Preview bảng từ JSON array hoặc CSV.

## Entry

`index.ts`

## Cách dùng

```bash
node dist/tools/table-preview/index.js --file data.csv --open
node dist/tools/table-preview/index.js --file data.json --format json
```

## Mô tả nhanh

- detect CSV/JSON
- render bảng có filter nhanh
- phù hợp để inspect dataset nhỏ và vừa
